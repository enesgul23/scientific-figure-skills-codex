# Journal Caption Rules

Use this reference when drafting captions, legends, notes, in-text references, and alt text.

## Caption Structure

1. Short descriptive title.
2. Panel-by-panel explanation.
3. Variables, groups, and units.
4. Sample sizes or denominators when relevant.
5. Statistical method, uncertainty type, and threshold notes.
6. Data source or model split notes when needed.
7. Style or mockup status when relevant.

## Rules

- Caption must not claim more than the figure shows.
- Do not infer causation from correlation.
- Do not call unverified journal style "compliant".
- State whether a graphic is design-only when data are missing.
- Avoid interpretive language that belongs in Results unless the figure directly supports it.

## Alt Text

Alt text should summarize chart type, axes, groups, main visible pattern, and important caveats. It should not repeat every data point unless the figure is simple.

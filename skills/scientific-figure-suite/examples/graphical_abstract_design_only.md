# Example: Graphical Abstract Design Only

## User Request

```text
fig-graphical-abstract Design a Science-style graphical abstract for a hydro-AI model coupling groundwater and surface flooding.
```

## Expected Behavior

- Route to `graphical-abstract` and `schematic-mechanism`.
- Mark visual elements as method-derived, evidence-derived, hypothesis, or illustrative.
- Avoid claiming the coupling causes performance gains unless supplied evidence supports it.
- Mark style as `ESTIMATED` or `UNVERIFIED` unless current Science guidance is supplied or checked.

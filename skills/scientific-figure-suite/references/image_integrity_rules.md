# Image Integrity Rules

Use this reference for microscopy, gels/blots, radiology, clinical images, satellite imagery, and other scientific images.

## Rules

- Do not alter image content in ways that change scientific meaning.
- Keep original image provenance when available.
- Apply global adjustments consistently and disclose them when relevant.
- Do not splice, clone, erase, or selectively enhance features.
- Preserve scale bars and modality metadata where relevant.
- For composite images, state channel mapping and merge rules.

## Audit Flags

- duplicated regions
- inconsistent backgrounds
- unexplained cropping
- missing scale bar
- missing acquisition metadata
- contrast adjustment not disclosed
- image panel lacks source or sample ID

## Output Language

If raw images are unavailable, state that image integrity cannot be fully verified. Do not declare image integrity `PASS` solely from a rendered figure.

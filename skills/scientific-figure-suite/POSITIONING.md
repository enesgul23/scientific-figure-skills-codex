# Positioning

## What This Is

Scientific Figure Suite is a Codex-native skill group for planning, creating,
auditing, captioning, styling, and packaging scientific figures. It targets
publication-quality scientific graphics, multi-panel figures, graphical
abstracts, model-performance figures, clinical figures, geospatial maps, omics
figures, visual claim ledgers, and reproducible export packages.

## What This Is Not

- Not an autonomous evidence generator.
- Not a journal compliance oracle.
- Not a replacement for domain review, statistician review, clinical review, or cartographic review.
- Not a web app, SaaS product, dashboard, or external service.
- Not a license to invent sample sizes, p-values, model metrics, biological mechanisms, clinical outcomes, or map attributes.

## Design Philosophy

**Evidence before aesthetics.** A beautiful figure that misstates evidence is a failed figure.

**Human-in-the-loop.** Users decide the scientific message, provide data, and verify final submission requirements.

**Failure visibility.** Missing data, unverified style, weak uncertainty, unsupported captions, and reproducibility gaps must be surfaced rather than hidden.

**Progressive disclosure.** The root skill routes; workflows and agent role prompts load only when needed.

## Submission Language

Use "journal-inspired estimated profile" unless current official or user-provided guidance has been checked in the active task. Use "submission-ready" only when data, code, style, caption, audit, and export gates are actually resolved.

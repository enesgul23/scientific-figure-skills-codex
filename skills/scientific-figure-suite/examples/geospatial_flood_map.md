# Example: Geospatial Flood Map

## User Request

```text
fig-plan Design a flood inundation map with DEM background, gauges, basin boundary, and model error overlay.
```

## Required Inputs

- CRS/projection for all layers
- DEM/raster path and units
- flood extent layer
- station/gauge coordinates
- basin boundary
- error variable and units
- basemap source if used

## Expected Gates

- G12 geospatial integrity is mandatory.
- No scientific map output should be approved without CRS/projection.

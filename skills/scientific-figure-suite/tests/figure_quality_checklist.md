# Figure Quality Checklist

Use this as a compact manual checklist for smoke testing.

| Gate | Required result |
|---|---|
| Data provenance | Data source or data absence is explicit |
| Visual claim ledger | Every result-implying element maps to support |
| Chart appropriateness | Chart type fits the scientific question |
| Statistical honesty | N, uncertainty, correction, and scales are handled |
| Axis and units | Axes, labels, units, and scale choices are visible |
| Caption integrity | Caption does not overclaim |
| Accessibility | Color, contrast, redundancy, and alt text considered |
| Journal status | `VERIFIED`, `ESTIMATED`, or `UNVERIFIED` is explicit |
| Export readiness | Formats and DPI match figure type |
| Reproducibility | Code and data pointers are present or design-only status is explicit |
| Image integrity | Raw image limitations are stated when applicable |

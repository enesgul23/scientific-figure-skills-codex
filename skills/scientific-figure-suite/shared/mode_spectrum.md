# Mode Spectrum

Use this to decide how much template material to load.

## Spectrum

| Position | Meaning | Figure-suite modes |
|---|---|---|
| Fidelity | Template-heavy, reproducible, checklist-driven | audit, caption, style, export |
| Balanced | Structured but adaptable | render, revision, full |
| Originality | Exploratory, dialogue-driven, template-light | planning, graphical abstract ideation |

## Loading Rule

- Fidelity modes may load templates and schemas early.
- Balanced modes load templates after the data/domain structure is known.
- Originality modes should start with questions and message design, not rigid output templates.

## Risk

Fidelity without evidence becomes false certainty. Originality without claim control becomes overclaiming. Use the visual claim ledger in both cases.

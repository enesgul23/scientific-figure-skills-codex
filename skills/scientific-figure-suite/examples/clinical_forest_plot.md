# Example: Clinical Forest Plot

## User Request

```text
fig-plan Prepare a Lancet-style forest plot from subgroup hazard ratios and 95% confidence intervals.
```

## Required Inputs

- subgroup names
- hazard ratios
- lower and upper CI
- group denominators
- endpoint definition
- follow-up window
- current Lancet guidance if verified compliance is requested

## Expected Warnings

- Style is not verified unless current guidance is supplied or checked.
- Denominators and endpoint definitions are required before final chart.

# AI Generated Figure Disclosure

Use this reference when AI-assisted design, AI-generated non-data artwork, or generated icons may be part of a figure.

## Rules

- Do not generate scientific evidence, raw data, clinical outcomes, biological mechanisms, or map attributes.
- AI-generated visuals may be used only as design mockups, schematic placeholders, or non-evidentiary illustration when clearly labeled.
- Data-derived panels must be generated from supplied data or code.
- Preserve a distinction between illustrative design elements and evidence-bearing elements.

## Suggested Labels

```text
NON_DATA_MOCKUP
DESIGN_ONLY
ILLUSTRATIVE_SCHEMA
EVIDENCE_DERIVED_PANEL
```

## Disclosure Pattern

```text
This design includes AI-assisted illustrative elements. Data-derived claims must be rendered from supplied data before submission use.
```

Do not add disclosure text into the figure unless the user or journal requires it; include it in the package notes or quality report.

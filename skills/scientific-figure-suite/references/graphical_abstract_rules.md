# Graphical Abstract Rules

Use this reference for graphical abstracts and visual summaries.

## Rules

- One primary message only.
- Separate method, evidence, and implication.
- Use minimal text and clear visual flow.
- Avoid decorative elements that imply unprovided data or mechanisms.
- Mark design-only outputs as `NON_DATA_MOCKUP`.
- Do not imitate journal logos, branded templates, or copyrighted figure styles.

## Narrative Patterns

- Problem -> method -> evidence -> implication.
- Input -> model/process -> output -> validation.
- Exposure -> biological response -> measured endpoint.
- Study population -> intervention -> outcome.

## Integrity Checks

- Are all entities supplied by the user?
- Are all arrows defined?
- Are hypotheses labeled?
- Does the visual metaphor change the scientific claim?
- Can the caption describe the graphic without inventing a result?
